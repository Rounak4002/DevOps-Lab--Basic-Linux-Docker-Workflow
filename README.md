# DevOpsLab - Basic Linux and Docker Workflow

This project demonstrates basic DevOps practices using Linux commands, shell scripting, and Docker.

## Features

* Basic Linux command usage
* Simple shell scripting automation
* Running containers using Docker

## How to Run

### Make script executable

```
chmod +x script.sh
```

### Run script

```
./script.sh
```

## What it does

* Displays system information
* Lists files in directory
* Runs an Nginx container using Docker
* Shows active containers

## Tech Stack

* Linux
* Shell Scripting
* Docker
